# Expense Tracker API

A RESTful API built with Flask and SQLite that allows users to manage expenses.

## Features
- Create, read, update, and delete expenses
- Persistent data using SQLite
JSON-bason API endpoint

## Endpoint

GET /expenses
POST /expenses
PUT /expenses/<id>

## Tech Stack
- Python
- Flask
- SQLite

## How to Run

'''bash
pip install -r requirements.txt
python app.py
